1. Item groups
    maxItems -- the maximum number of items the adaptive algorithm should administer from this group (passage). This is subject to each item's 'isRequired' flag.
    1. numRequired
      This parameter do not figure into the item selection process. It determines how many items a student must respond to and is affected by the value of every item's isRequired flag within the group.
      1. numrequired = -1
        All items in the group (that have been administered subject to strict max and maxItems pruning) must be responded to by the student
      2. numRequired > -1
        1. Student must respond to every required item
        2. if the number of required items < numRequired, student must respond to additional optional items until the number of responses >= numRequired.
