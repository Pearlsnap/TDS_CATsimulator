1. Segmented test considerations
    1. Segment blueprints
      Each segment's blueprint is evaluated in total isolation from its 'siblings'.
      1. Number of items
        1. Min/max
          So the minimum number of items on any strand or content level is observed for that segment alone.
        2. Number administered
          The number of items considered administered on a segment is only for that segment.
        3. Item masking
          Any itemgroup selected for any previous segment is removed from the pool for all subsequent segments. An itemgroup may be administered at most once on a test.
      2. Strict max
        The strict max is observed for the segment only. A test (all segments combined) is considered to have a strict max on a content level only if every instance of that content level on every segment is a strict max.
    2. Intersecting item pools
      Items may appear in more than one segment, but will be administered at most once on any give test.
    3. Heterogeneous segments
      Because item selected is predetermined and immutable on fixed form segments, some rules must apply.
      1. Adaptive and fixed form
        1. Adaptive preceding
          If an adaptive segment precedes any fixed form segment, their item pools must be non-intersecting. That is, no item in the adaptive pool may exist in the fixed form pool.
        2. Fixed form preceding
          If an adaptive segment follows a fixed form segment, then its item pool may intersect with the fixed form item pool. That is because the adaptive segment will not select an item that has already been selected for a test.
      2. Successive fixed form segments
        1. Form cohorts
          A segmented test in which each segment is a fixed form test often started out as a single fixed form test with multiple forms which was then divided into segments.
          It is critical to keep the segments of the original form together so they are labeled with a 'cohort' designator. Form segments with a common cohort designator are kept together on a test.
          For this reason the following hold
          1. Items may not be duplicated on forms within a given cohort
          2. Items may be duplicated on forms NOT on the same cohort
          3. The rules for mixing multiple form cohorts with adaptive segments still apply (adaptive preceding and succeeding).
