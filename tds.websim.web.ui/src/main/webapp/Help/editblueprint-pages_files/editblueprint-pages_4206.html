1. Items
    isFieldTest -- for informational purposes only.
    isActive -- determines whether the item on an adaptive or independent field test will be administered. Not applicable to fixed form tests.
    isRequired -- if true, then the item may not be removed from an itemgroup by the adaptive algorithm to constrain the blueprint. In 'real life' it also means the student must answer the item (but, in passages this is best controlled with the 'numrequired' parameter.
    Rule of thumb: for items in passages set this to false (0) and the passage 'numrequired' to -1 (meaning all items must be responded to)
    For items not in passages set this to true (1) unless the item really is optional for all students.
